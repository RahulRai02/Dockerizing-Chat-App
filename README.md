
# Dockerized Chat App


## How to run?

To run the dockerized version, run:

```bash
  docker-compose up -d --build
```

To run undockerized version:

```
cd server
node ./server.js
```

```
cd client
node ./client.js
```
Create multiple instances of the same client to generate multiple users.
